# Medical-Edge-LLM

A portfolio project that demonstrates the complete workflow of adapting, compressing, and deploying a domain-specific Large Language Model (LLM) for offline medical question answering.

The project follows an end-to-end AI engineering pipeline—from dataset generation and QLoRA fine-tuning to model quantization and deployment on resource-constrained devices.

---

## Project Overview

The objective is to transform an open-source language model into an efficient medical assistant capable of running offline on standard hardware while maintaining good response quality.

The project covers:

- Medical dataset generation using a teacher model
- Parameter-efficient fine-tuning with QLoRA
- Model optimization through post-training quantization
- Local inference on Windows
- Deployment preparation for Android devices
- Performance benchmarking and analysis
